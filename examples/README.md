# Example Custom Bouncers

This directory contains example implementations of custom bouncers to help you create your own.

## Examples

- **jupyter_bouncer.py** - Complete Jupyter notebook bouncer implementation

## Usage

These are reference implementations. To use them:

1. Copy the example to the `bouncers/` directory
2. Add configuration to `bouncer.yaml`
3. Register in `bouncers/__init__.py`

See [CREATING_BOUNCERS.md](../docs/CREATING_BOUNCERS.md) for detailed instructions.
